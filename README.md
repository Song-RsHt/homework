# 智能灯光控制系统

### 项目目标

设计并实现一个基于命令行的智能灯光控制系统，支持灯光状态切换、定时控制、亮度与色温调节等功能，提升用户对灯光设备的便捷管理体验。

### 核心功能

一.基础控制

1.支持灯光的开关状态切换。

2.可调节灯光亮度（范围：1-100）和色温（范围：1850-6500K）。

二.定时功能

1.根据当前灯光状态（开/关），设置未来时间自动切换状态（如：灯光关闭时设置定时开灯，灯光开启时设置定时关灯）。

2.输入分钟数后，系统自动执行对应操作。

三.用户交互

1.提供命令行界面，引导用户选择操作（返回主界面、退出程序、查看帮助等）。

2.输入验证机制，确保时间、亮度等参数符合有效范围。

### 技术实现

开发语言：C语言

数据结构：通过 LightZone 结构体管理灯光区域的状态（开关、亮度、色温等）。

核心函数：

next()：处理用户交互逻辑，提供操作选项。

sensor(int i)：执行灯光状态切换、亮度/色温调节。

timing()：实现定时控制功能，支持未来时间自动切换灯光状态。

show_time()：显示当前时间（需结合实际时间获取逻辑）。

### 应用场景

1.家庭或办公环境中，用户可通过命令行快速管理灯光设备。

2.支持定时任务，满足用户在特定时间自动控制灯光的需求（如睡前定时关灯）。
